# PytestAutotest

#### 介绍

PytestAutotest是有个集UI+API自动化与一体的自动化测试项目，通过主动的编写pytest的测试函数，pytest+allure框架设计的UI和API的自动化测试框架，可以读取sqlite或飞书共享文档中的项目，测试环境，通知加上用例信息，来执行测试用例。飞书共享文档可以让更多的人参与进行协作！并生成美观的测试报告，可以发送给相关人员进行通知！
